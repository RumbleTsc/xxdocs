# XXDocs

> XXDocs，也叫「XX文档」，核心定位是「全栈开发体系化文档」

## 简介

本项目旨在打造一本开源免费、新手友好的全栈开发参考文档，核心特色：

- 全栈：文档全面，覆盖前端、后端、移动端、跨端等多端开发相关技术
- 体系：内容体系化，拒绝知识点零散+碎片化，从基础到进阶，构建完整知识体系
- 实用：重点突出+简单易读，方便开发入门、日常参考、知识点回顾

**ps：目前XXDocs正在持续完善中，欢迎一起共建，[官网](https://docs.xxhelper.com/)上部分技术暂时跳转对应官方文档。**

## 关于全栈

传统的全栈（Full Stack）一般指Web全栈，即Web前端+后端。

XXDocs定位里提到的全栈不局限于传统的Web全栈，也包括移动端、跨端等相关内容。

不管只是出于兴趣简单了解别的技术栈，还是想体系化学习成为软件开发里的T型人才，亦或是立志成为独立开发者，希望XXDocs都能成为你的重要参考。
